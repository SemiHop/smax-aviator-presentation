## File Format Requirements

- **Format:** PNG (Portable Network Graphics)
- **Recommended Resolution:** 1920x1080 or higher
- **File Size:** Keep under 2MB per image for optimal presentation performance
- **Aspect Ratio:** 16:9 works best for most slides

## Tips for Great Screenshots

1. **Clear & Uncluttered:** Remove unnecessary UI elements or test data
2. **Highlight Key Areas:** Use rectangles, arrows, or highlights to draw attention
3. **Consistent Styling:** Use similar zoom levels and UI themes
4. **Real Data:** Use realistic (but anonymized) data, not "test test test"
5. **High Contrast:** Ensure text is readable when projected
6. **Annotations:** Add callouts or labels to explain important features

## Testing Your Screenshots

After replacing placeholders:
1. Open `smax-aviator-presentation.html` in your browser
2. Navigate to each slide with screenshots
3. Verify images display correctly
4. Check image clarity and readability
5. Ensure all screenshots maintain consistent quality
