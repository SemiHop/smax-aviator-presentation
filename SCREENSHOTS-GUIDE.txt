HOW TO REPLACE
--------------
OPTION 1: Overwrite the file
  - Save your screenshot with the same filename
  - Copy it to the screenshots folder
  - It will replace the placeholder automatically

OPTION 2: Rename your file
  - Rename your screenshot to match the placeholder name
  - Copy it to the screenshots folder

EXAMPLE

If you have a screenshot called "my-architecture.png":
